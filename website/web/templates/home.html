MISP Project

CTI Transmute

CTI-Transmute is a tool for converting Cyber Threat Intelligence (CTI) data between MISP and STIX formats. It provides a set of API endpoints that allow automated conversion of data, making it easier to integrate different threat intelligence platforms and workflows. This project is maintained by the MISP Project community and aims to standardize CTI data exchange.
